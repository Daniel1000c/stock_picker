# 📈 Stock Picker

A Ruby program that determines the best days to buy and sell a stock in order to achieve the maximum possible profit.

This project was completed as part of the **Ruby course** from **The Odin Project**.

---

## 🧠 How It Works

The method takes an array of stock prices where each index represents a day.
It then returns the best day to buy and the best day to sell for the highest profit.

Example:

```ruby
stock_picker([17,3,6,9,15,8,6,1,10])
```

Output:

```ruby
[1, 4]
```

Explanation:

* Buy on day **1** at price **3**
* Sell on day **4** at price **15**
